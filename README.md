# QRcodedasorte

Projeto de mensagens diárias no estilo "biscoito da sorte", acessadas via QR Code.

## Como usar

1. Instale as dependências:
```
npm install
```

2. Inicie o servidor:
```
npm start
```

3. Gere o QR Code:
```
npm run qr
```

4. Acesse:
- Página principal: `http://localhost:3000`
- Admin: `http://localhost:3000/admin.html`
